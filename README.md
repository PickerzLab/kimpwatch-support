# 오늘은얼마 지원 페이지

환율이 원하는 가격에 올 때까지 매번 확인하고 계신가요?
오늘은얼마는 주요 통화와 USDT, 김치프리미엄을 한눈에 보여주고, 설정한 감시가에 도달하면 알려드립니다.

 

## 앱 정보

* **앱 이름**: 오늘은얼마
* **카테고리**: 금융
* **개발자**: 피커즈
* **지원 언어**: 한국어

 

## 주요 기능

* **8개 통화**: USD, JPY, EUR, GBP, AUD, CAD, HKD, CNY 환율 확인
* **5가지 환율**: 매매기준율, 송금 보낼 때, 송금 받을 때, 현찰 살 때, 현찰 팔 때
* **우대환율**: 개인 우대율을 반영한 예상 환율 계산
* **환율 타임머신**: 최근 환율과 1년·3년 평균 비교
* **USDT**: 업비트 KRW-USDT 시세와 24시간 변동 확인
* **김치프리미엄**: USDT와 USD 매매기준율을 비교한 국내외 가격 차이 확인
* **감시가 알림**: 환율·USDT·김치프리미엄이 설정한 가격에 도달하면 알림
* **홈 화면 위젯**: 앱을 열지 않고 주요 시세 확인

 

## 자주 묻는 질문

**한국수출입은행 환율은 어떻게 이용하나요?**
앱 설정에서 한국수출입은행 Open API 키를 입력하면 공식 매매기준율을 이용합니다. API 키가 없으면 유럽중앙은행 환율로 대체됩니다.

**주말과 공휴일에도 환율이 갱신되나요?**
한국수출입은행 환율이 발표되지 않는 날에는 최근 영업일 데이터를 표시합니다. USDT와 김치프리미엄은 이용 가능한 최신 시세를 기준으로 갱신됩니다.

**감시가에 도달했는데 알림이 늦게 왔어요.**
현재 알림은 앱이 백그라운드에서 시세를 확인한 뒤 보내는 방식입니다. iOS의 배터리 관리, 저전력 모드, 백그라운드 앱 새로 고침 상태에 따라 알림이 늦어지거나 표시되지 않을 수 있습니다.

**표시된 송금·현찰 환율은 실제 은행 환율과 같나요?**
선택한 통화의 매매기준율과 수수료율, 우대율을 바탕으로 계산한 예상값입니다. 실제 적용 환율은 은행과 거래 시점에 따라 달라질 수 있습니다.

**김치프리미엄은 어떻게 계산하나요?**
업비트 KRW-USDT 가격을 USD 매매기준율과 비교해 계산합니다. 거래소 시세와 환율 갱신 시점에 따라 실제 시장 가격과 차이가 날 수 있습니다.
